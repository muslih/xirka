UTAG XU186 - NFC Memory Card Chip

UTag XU168 is member of the UTag family device from Xirka Silicon Technology which is designed specifically for memory card chip application. UTag XU168 fully complies with communication layer (RF Interface) ISO/IEC14443 Type A parts 2 and 3 specifications, NFC Forum Type 2 Tag technical specifications (see Ref. 2 and Ref. 3) and NDEF data structure configurations.

The security of the family device is accomplished in hardware with not only protection against out of parameter operation of the device, but also hardware memory management to protect against security attacks. Features of the UTag family device make them particularly suitable in smart advertisement, connection handover, gaming, product authentication, and mobile companion tags.

Interface

UTag XU168 is member of UTag family complies with the following industry standards and requirements:

  1. ISO/IEC 14443-2 (Type A) Compatible.
  2. ISO/IEC 14443-2 (Type A) Compatible.
  3. NFC Forum Tag 2 Type technical specification and NDEF data structure configurations.
